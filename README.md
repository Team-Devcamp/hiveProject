# HIVE(의류 쇼핑몰 프로젝트)

## 프로젝트 소개
'HIVE'는 직관적인 UI와 다양한 사용자 편의기능을 통해 남녀노소 누구나 간편하게 쇼핑을 즐길 수 있는 의류 쇼핑몰 프로젝트 입니다.
>*웹사이트 주소 : *

## 프로젝트 일정 및 규모
### 프로젝트 일정
* 총 개발 기간 : 16일(2022.7.11 ~ 7.26)
* 총 개발 시간 : 192H(12H x 16일)
* 1인당 일일 평균 개발 시간 : 10H
* 인원 : 6명(멘토 1명 포함)

### 프로젝트 규모(기준일 : 2022.9.10)
* 파일 204개
* 코드 14,718줄
* 모듈 13개
* JAVA 파일, 코드 비율 49%, 37%

>*세부 JAVA, JSP, CSS, 기타 규모는 제일 하단 표 참고*
